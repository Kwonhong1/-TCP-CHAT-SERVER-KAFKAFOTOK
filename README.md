# -TCP-CHAT-SERVER-KAFKAFOTOK

공군의 일원으로서 지내고 있는 현재 만드는 프로젝트이기에 ROKAF에서 이름을 따 KAFKAFOTOK으로 지었습니다.

상용을 목적으로 한다거나, 포트폴리오를 위해 제작하는 프로젝트는 아니나, 이론만 공부하다 보니 점점 개발자로부터 멀어지는 기분이 들어 시작하게 되었습니다.

**주요 기능은 다음과 같습니다.**
1. 채팅 기능
  - 평범한 채팅 기능입니다.
  - 카카오톡처럼 여러 방, 여러 친구들과 채팅을 진행할 수 있도록 합니다.
2. '마이룸'
  - 과거 싸이월드 속 내 집을 꾸미듯 인테리어 해 채팅방의 요소로 사용합니다.
  - 실시간 상호작용을 더함으로써, 데이터베이스와 통신 기능 최적화를 학습해볼 예정입니다.
3. 이모티콘, 채팅방을 덮은 이모지
  - 카카오톡에서 국가적인 행사가 있을 때 사용하는 특정 메시지를 입력하면 해당 메시지와 관련된 이모지가 나오는 것을 구현해보려 합니다.
